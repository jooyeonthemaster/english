# 학생용 듀오링고 스타일 내신 학습 서비스 — 기획서

> 담당: 김제연
> 최종 업데이트: 2026-03-25
> 상태: 기획 완료, 구현 전

---

## 1. 서비스 개요

### 한줄 요약
내신 시험 대비를 위한 **지문 반복 노출 자율학습 도구** (듀오링고 스타일 모바일 앱)

### 포지셔닝
- **숙제/시험을 대체하지 않음** → 자율학습 보조 도구
- 자투리 시간에 유튜브 대신 하는 느낌 (최소 3분)
- 강사가 내신 기간/지문을 설정, 학생은 앱만 열면 자동으로 맞는 콘텐츠 노출
- 내신 전문 학원이 주 타겟 (첫 테스트 학원: 강동구 내신 전문)

### 듀오링고와의 핵심 차이

| | 듀오링고 | 우리 서비스 |
|--|---------|-----------|
| 시간 구조 | 무한 연속 | **시험 단위 사이클** (데드라인 존재) |
| 콘텐츠 수명 | 영구 누적 | **시험 끝나면 아카이브** |
| 지문 길이 | 2~3문장 | **문단 수준** (200~400단어, 10~20문장) |
| 학습 동기 | 자기주도 | **시험 점수** (외부 동기) |
| 커리큘럼 | AI 자동 | **강사가 지정** (내신 모드) |
| 문제 유형 | 듣기/말하기/읽기/쓰기 | **읽기 + 서술형만** |

### 듣기/말하기 제외 이유
- **듣기** → 교육청 별도 국가시험 (연 4회), 학원 대비 영역 아님
- **말하기** → 수행평가(발표/스피치), 앱으로 연습 불가
- 내신 지필고사 = 선택형 70~80% + 서술형 20~30%
- 수행평가(내신 30~40%)는 별도 대비 모드 불필요 → 세션 학습이 어휘/문법/교과서 숙달로 자연스럽게 연결

## 2. 듀얼 모드 시스템

### 내신 집중 모드 (강사가 ON/OFF)
- 강사가 **기간 설정** (보통 시험 3~4주 전 시작, 기본 4주)
- 시험 범위 **지문 지정** (8~15개)
- 학생 앱에 **내신 지문만** 노출
- **데드라인 카운트다운** 표시 ("시험 D-14")
- 학습 스케줄 **D-day 역산** 자동 계산
- 대비 기간은 강사가 3주/4주/5주 조절 가능

### 평상시 모드 (기본)
- **학원 커리큘럼과 완전 별도** (자율학습이므로 강제성 없음)
- 학년/수준 기반으로 **시스템이 지문 자동 추천**
- 수능 기출, 모의고사, 다양한 주제 지문
- 강사는 평상시 모드에 관여할 필요 없음 (부담 최소)
- XP/랭킹/스트릭은 동일하게 적용

### 평상시 모드를 커리큘럼과 분리한 이유
- 자율학습이라 강제성 없음 → 학생마다 진도가 천차만별
- 학원 수업에서 이미 커리큘럼대로 배우고 있음
- 앱은 추가 학습이지 수업 대체가 아님
- 강사가 내신 + 평상시 다 관리하면 부담 2배

### 내신 콘텐츠 소진 시
내신 지문 레슨을 전부 완료한 학생:
```
1순위: 오답 집중 복습 (내신 지문 약점)
2순위: Stories 재읽기
3순위: 종합 모의 세션 (내신 지문 믹스)
  ↓ 전부 소진 / 숙달도 충분
4순위: 평상시 콘텐츠 자동 노출
       "내신 준비 완료! 수능 기출도 풀어볼래?"
       → 수능/모의고사 지문 세션 (XP/랭킹 동일 적용)
```
- AI 추가 문제 생성 불필요 (비용 절감 + 같은 지문 300개 이상은 질 저하)
- 새로운 지문으로 넘어가는 게 학습 효과도 더 좋음

## 3. 계층 구조

```
시즌 (내신 집중 기간 or 평상시)
├── 레슨 1 = 지문 A
│   ├── 세션 1: 어휘 중심 (15문제, ~3분)
│   ├── 세션 2: 해석 중심 (15문제, ~3분)
│   ├── 세션 3: 문법 중심 (15문제, ~3분)
│   ├── 세션 4: 종합 믹스 (15문제, ~3분)
│   ├── 세션 5: 틀린 부분 집중 (15문제, ~3분)
│   └── 📖 Stories: 전체 지문 통독 (~5분)
│
├── 레슨 2 = 지문 B (동일 구조)
├── ...
└── 레슨 N = 지문 N
```

**핵심 원칙: 지문 1개 = 레슨 1개** (지문이 여러 레슨에 흩어지지 않음)

## 4. 세션 구조 (핵심 학습 단위)

### 기본 스펙
- **1세션 = 15문제, ~3분** (문제당 평균 10~15초)
- **1레슨 = 5세션 + Stories**
- **지문 전체에서 출제** (청크 분할 X → 매 세션마다 지문 전체 노출)

### 세션별 유형 비중 (전체 지문 대상)

| 세션 | 주 유형 | 구성 |
|------|---------|------|
| 세션 1 | **어휘 중심** | 어휘 10 + 해석 3 + 문법 2 |
| 세션 2 | **해석 중심** | 해석 10 + 어휘 3 + 문법 2 |
| 세션 3 | **문법 중심** | 문법 10 + 어휘 3 + 해석 2 |
| 세션 4 | **종합 믹스** | 어휘 5 + 해석 5 + 문법 3 + 추론 2 |
| 세션 5 | **약점 집중** | 이전 틀린 유형 위주 + 남은 유형 |

### 문제 유형 상세

**어휘 (내신 빈출)**
- 뜻 매칭 (영→한, 한→영)
- 문맥상 의미 추론
- 동의어/반의어

**해석**
- 문장 배열 (단어 → 올바른 순서)
- 해석 선택 (4지선다)
- 빈칸에 알맞은 표현

**문법 (내신 출제 비중 최대)**
- 어법 판단 (맞는/틀린 것 고르기)
- 틀린 부분 고쳐 쓰기 (직접 입력 — 단어/구 수준)
- 문법 포인트 적용

**종합/추론 (후반 세션)**
- 빈칸 추론
- 문장 삽입
- 순서 배열

### 난이도 설계
- 세션별 난이도 차등 **없음** (비슷한 난이도로 유형만 다르게)
- AI한테 "쉬운/어려운" 구분보다 "어휘/문법" 구분이 결과가 일관적
- 반복의 핵심은 난이도 상승이 아니라 **다양한 각도에서 같은 지문 노출**

### 반복 노출 효과
- 세션 5개 = 지문 전체 **5회 노출**
- Stories = 추가 **1회 통독**
- 복습 세션 = 추가 **2~4회**
- **총 8~10회 반복 노출** (내신 대비에 충분한 수준)

## 5. Stories 모드 (지문 통독)

레슨(세션 5개) **완료 후 언락**되는 전체 지문 읽기 모드.
듀오링고 Stories 방식 차용 — 긴 지문을 모바일에서 자연스럽게 처리.

### UX 플로우
```
문장 1 표시 → (탭하면 해석/어휘 팝업)
문장 2 표시
문장 3 표시
💬 이해도 체크 문제 (가볍게)
문장 4 표시
문장 5 → 핵심 단어 하이라이트
문장 6 표시
💬 문법 체크 문제
...
마지막 문장
💬 전체 주제/요약 문제
```

### 인터랙션
| 동작 | 기능 |
|------|------|
| 단어 탭 | 뜻 + 품사 + 예문 팝업 |
| 문장 탭 | 해석 표시 (토글) |
| 밑줄 문법 | 문법 포인트 설명 팝업 |
| 중간 문제 | 3~5개, 이해도 체크 (가볍게) |
| 스크롤 | 문장이 하나씩 아래로 추가되며 진행 |

### 세션 vs Stories 비교
| | 세션 (문제 풀이) | Stories (지문 읽기) |
|--|-----------------|-------------------|
| 목적 | 세부 포인트 학습 | 전체 통독 + 맥락 파악 |
| 분량 | 지문 전체 (유형별) | 지문 전체 (순서대로) |
| 문제 | 15개 (집중적) | 3~5개 (가볍게) |
| 느낌 | 공부 | 읽기 (몰입) |
| 시간 | ~3분 | ~5분 |
| 용도 | 학습 | 복습 + 통합 |

### Stories 반복
- 완독 후에도 재읽기 가능
- 매번 **다른 위치에 다른 문제**가 끼어듦
- 시험 직전 빠르게 훑기용

## 6. 문제 생성 방식

### 사전 대량 생성 (듀오링고와 동일 방식)

```
강사가 지문 등록
  → AI가 유형별로 문제 대량 생성 (1회)
     ├── 어휘 문제 60~80개
     ├── 해석 문제 60~80개
     ├── 문법 문제 60~80개
     └── 종합/추론 40~60개
     = 지문당 총 200~300개 후보
  → DB에 저장
  → 강사가 검수 (선택적)
  → 학생 학습 시 DB에서 선별 출제
```

### 출제 로직 (코드로 처리, AI 불필요)
```
세션 "어휘 중심" → DB에서 어휘 10개 + 해석 3개 + 문법 2개 랜덤 추출
세션 "문법 중심" → DB에서 문법 10개 + 어휘 3개 + 해석 2개 랜덤 추출
복습 세션 → 이전 틀린 문제 + 같은 유형 안 푼 문제 믹스
```

### 듀오링고 참고 수치
- 듀오링고: 레슨당 후보 ~200개, 14~15개 선별 출제
- 우리: 지문당 200~300개, 세션당 15개 (5세션 = 75문제 소모)
- 여유분 125~225개 → 복습/실전에 충분
- 소진 시 AI 추가 생성 대신 **평상시 콘텐츠로 전환**

### AI 비용 최적화
- API 호출은 **지문 등록 시 1회만**
- 학생 학습 시에는 DB 조회만 (빠르고 무료)
- 같은 지문을 공부하는 모든 학생이 같은 문제 풀 공유
- 실시간 AI 추가 생성 로직 불필요 → 개발 단순화

## 7. 4주 학습 스케줄 (자동 역산)

### 기본 전제
- 내신 대비 기간: **4주 (기본)**, 강사가 3~5주 조절 가능
- 시험 범위: 지문 **8~15개** (교과서 + 외부)
- 하루 목표: **2~3세션 (6~9분)**

### 지문 10개 기준 스케줄

| 주차 | 기간 | 앱 학습 내용 | 하루 분량 |
|------|------|-------------|----------|
| 1주차 | D-28~D-22 | 지문 1~3 세션 학습 | 2세션 (~6분) |
| 2주차 | D-21~D-15 | 지문 4~7 + 1~3 복습 | 2~3세션 (~9분) |
| 3주차 | D-14~D-8 | 지문 8~10 + Stories 시작 | 2~3세션 |
| 4주차 | D-7~D-1 | 전체 복습 + 취약 집중 | Stories + 약점 세션 |

### 수치 검증
```
지문 10개 × 세션 5개 = 50세션 + Stories 10개 = 60개 학습 단위
28일 × 2.5세션 = 70세션 → 여유 있음
```

### 자동 조절
- 기간이 짧으면 (3주) → 하루 세션 수 자동 증가
- "오늘의 추천 학습량" D-day 역산 자동 계산
- 숙달도 낮은 지문 우선 배치

```
15문제 완료 → "+15 XP 획득!"

           → 계속하면 추가 XP
```

## 9. 데일리 미션

매일 2개 (쉬운 거 1개 + 어려운 거 1개)

```
오늘의 미션

🟢 세션 1개 완료하기
   보상: 경험치 1.2배 (10분간)

🔴 세션 3개 완료 + 정답률 80% 이상
   보상: 경험치 2배 (10분간)
```

- 보상 = XP 직접 지급이 아니라 **배율 버프 (10분 제한)**
- 미션 달성 후 바로 추가 학습 유도
- 시스템 자동 생성 (강사 관리 불필요)

## 10. 랭킹 시스템 (3단계)

### 구조: 3탭 (개인 / 학교 / 학원)

```
📊 랭킹
[개인]  [학교]  [학원]

상위 5위 노출
─────────────────────
1위 🥇 ***     520 XP
2위 🥈 ***     480 XP
3위 🥉 ***     450 XP
4위    ***     420 XP
5위    ***     390 XP
─────────────────────
          ...
142위  나      180 XP  ← 상위 5 밖이면 여기에 본인만 표시
─────────────────────
```

| 랭킹 | 범위 | 기준 | 리셋 |
|------|------|------|------|
| **개인** | 전체 유저 | 주간 XP | 매주 월요일 |
| **학교** | 전체 학교 | 학생 **평균** XP | 매주 월요일 |
| **학원** | 전체 학원 | 학생 **평균** XP | 매주 월요일 |

### 전체 유저 대상인 이유
- XP는 학습량 기반이라 시험 범위 달라도 공정
- 반/학원 단위는 학생 수가 적어서 경쟁 안 됨

### 학교/학원이 평균인 이유
- 총합이면 대형 학교/학원만 상위권 독점
- 평균이면 소규모도 경쟁 가능
- "학생 평균 학습량 1위"가 마케팅에 더 강력

### 향후 확장 (유저 증가 시)
- 초기: 단순 전체 순위표
- 성장기: 듀오링고식 리그 시스템 (Bronze→Diamond)

## 11. 내신 시험 참고 정보

### 시험 범위 (고등학교)
- 교과서: 한 학기 5개 레슨 (능률, 미래엔, 동아, 천재, YBM 등)
- 중간고사: 2~3개 레슨
- 기말고사: 2~3개 레슨
- 총 지문: 교과서 + 외부(EBS/모의고사) = **8~15개**

### 중학교
- 범위: 교과서 2~3개 단원, 외부 지문 거의 없음
- 총 지문: 4~6개

### 지필고사 문제 구성
- 선택형 70~80% + 서술형 20~30%
- **문법 출제 비중 최대**
- 빈칸 추론, 문장 삽입, 순서 배열, 어법 판단

### 대비 기간
- **4주가 표준** (최근 트렌드)
- 학원들은 시험 30문제 대비로 변형 문제 1,000개+ 준비
