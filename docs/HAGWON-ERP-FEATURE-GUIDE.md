## 1. 주요 경쟁 플랫폼 분석

### 종합 관리형 (ERP 성격)

| 플랫폼 | 핵심 특징 | 강점 |
|--------|----------|------|
| **학원스타 / 학원탑** | 학원 전용 종합 ERP | 수납, 출결, 반편성, 성적, SMS, 상담, 회계/세무 연동 |
| **클래스업** | 모바일 중심 학원 관리 | 출결 알림, 알림장, 수납, 상담 기록 |
| **학원조아** | 올인원 관리 | 출결, 수납, 성적, 상담, SMS |
| **어나더클래스** | 클래스 관리 특화 | 반편성, 시간표, 교재 진도 |
| **아카데미업** | 중소 학원 타겟 | 가벼운 UI, 기본 관리 기능 |

### 소통 특화형

| 플랫폼 | 핵심 특징 | 강점 |
|--------|----------|------|
| **클래스팅** | 학부모-교사 소통 → 학원 관리 확장 | 직관적 UI, 모바일 중심, AI 학습 분석 |
| **하이클래스** | 학교/학원 겸용 커뮤니케이션 | 알림장, 설문, 투표, 사진, 일정 — 학부모 친숙도 높음 |
| **아이엠스쿨** | 학부모 앱 기반 | 출결 알림, 알림장, 학원 홍보, 학부모 리뷰 |

### 특수 기능형

| 플랫폼 | 핵심 특징 | 강점 |
|--------|----------|------|
| **출결버스** | 출결 전문 | QR/NFC/GPS 기반, 학부모 실시간 알림 |
| **결제선생** | 수납 전문 | CMS 자동이체, 카드 정기결제, 미납 관리 |
| **통통통** | SMS/알림톡 전문 | 대량 발송, 카카오 연동, 템플릿 |
| **빡공시계** | 학습 시간 관리 | 공부 타이머, 랭킹, 자기주도학습 동기부여 |

### 시장 공백 (NARA의 기회)

대부분의 경쟁 플랫폼은 **행정 관리(출결, 수납, 알림)**에 집중하고 있으며, 다음 영역에서 뚜렷한 공백이 존재:
- AI 기반 학습 콘텐츠 생성 + ERP 통합
- 심층 학습 분석 (영역별 강약점, 오답 패턴)
- 게이미피케이션 기반 학생 동기부여
- 원장 경영 인텔리전스 (이탈 예측, 수익성 분석)

## 3. 수납 관리 상세

### 3.1 핵심 워크플로우

```
수강 등록
  → 수강료 산정 (일할 계산, 할인 적용)
  → 청구서 생성
  → 수납 (자동이체/카드/현금/간편결제)
  → 영수증 발행
  → 미납 시 독촉 프로세스
```

### 3.2 수납 방법

| 방법 | 설명 | 비고 |
|------|------|------|
| CMS 자동이체 | 매월 지정일 자동 출금 | 가장 보편적 |
| 카드 정기결제 | 매월 카드 자동 결제 | PG사 연동 필요 |
| 계좌이체 | 수동 확인 필요 | 소규모 학원 다수 사용 |
| 현금 수납 | 현금 직접 수령 | 현금영수증 의무 발행 |
| 간편결제 | 네이버페이, 카카오페이 | 청구서 링크 발송 → 결제 |

### 3.3 세부 기능

| 기능 | 설명 |
|------|------|
| **수강료 자동 계산** | 월 수강료, 일할 계산 (중도 등록 시 잔여일 기준) |
| **다과목 통합 청구** | 복수 과목 수강 시 하나의 청구서로 |
| **할인 정책 관리** | 형제 할인, 장기 등록 할인, 소개 할인, 얼리버드 |
| **미납 단계별 관리** | 1차 알림 (D-3) → 2차 알림 (D+3) → 전화 상담 (D+7) → 수강 제한 (D+14) |
| **환불 자동 계산** | 학원법 시행령 제18조 기준 (아래 상세) |
| **교재비/재료비** | 수강료와 별도 수납 항목 |
| **수납 내역 조회** | 학부모가 앱에서 직접 확인 |
| **납입증명서 발급** | 연말정산용 교육비 납입증명서 |

### 3.4 환불 계산 규정 (학원법 시행령 제18조)

```
수업 시작 전
  → 이미 납부한 수강료 전액 환불

수업 시작 후
  → 총 수업시간 1/3 경과 전: 수강료 2/3 환불
  → 총 수업시간 1/2 경과 전: 수강료 1/2 환불
  → 총 수업시간 1/2 경과 후: 환불 불가
```

### 3.5 주요 재무 지표

| 지표 | 산식 | 용도 |
|------|------|------|
| 월 매출 | 수강료 + 교재비 + 기타 수입 | 기본 경영 지표 |
| 수납률 | 실 수납액 / 청구액 × 100 | 수금 효율 |
| 학생당 매출(ARPU) | 월 매출 / 재원생 수 | 단가 분석 |
| 반별 수익성 | (반 매출 - 강사비 - 교재비) / 반 매출 | 수업 효율 |
| 미납률 | 미납액 / 청구액 × 100 | 재무 리스크 |

## 4. 출결 관리 상세

### 4.1 출결 상태 종류

| 상태 | 설명 |
|------|------|
| 출석 (PRESENT) | 정상 출석 |
| 결석 (ABSENT) | 무단/사유 결석 |
| 지각 (LATE) | 수업 시작 후 도착 |
| 조퇴 (EARLY_LEAVE) | 수업 종료 전 퇴원 |
| 보강 (MAKEUP) | 결석 보충 수업 참여 |
| 휴원 (ON_LEAVE) | 장기 휴원 중 |

### 4.2 체크인 방식

| 방식 | 장점 | 단점 |
|------|------|------|
| QR 코드 | 비용 낮음, 스마트폰 활용 | 대리 출석 가능 |
| NFC 카드 | 빠르고 정확 | 카드 분실, 하드웨어 비용 |
| 키오스크 터치 | 직관적 | 기기 비용 |
| 교사 수동 입력 | 가장 정확 | 시간 소요 |

### 4.3 자동화 알림

| 이벤트 | 수신자 | 채널 |
|--------|--------|------|
| 학생 도착 | 학부모 | 푸시/알림톡 |
| 학생 퇴원 | 학부모 | 푸시/알림톡 |
| 무단결석 감지 | 학부모 + 교사 | 푸시/SMS |
| 지각 도착 | 교사 | 앱 내 알림 |
| 출석률 저하 경고 | 원장 | 대시보드 알림 |

### 4.4 출결 통계

- 학생별 월/학기 출석률
- 반별 평균 출석률
- 요일/시간대별 결석 패턴
- 출석률 하위 학생 리스트 (이탈 위험 지표)

## 5. 반/수업 편성 상세

### 5.1 반 관리

| 기능 | 설명 |
|------|------|
| 반 생성 | 이름, 과목, 레벨, 요일/시간, 교실, 담당 강사, 정원 |
| 학생 배정 | 레벨 테스트 결과 기반 배정 |
| 정원 관리 | 신고 정원 초과 방지 경고 |
| 반 이동 | 레벨업/레벨다운 시 반 변경 이력 기록 |
| 대기 명단 | 정원 초과 시 대기 등록 |

### 5.2 시간표 관리

| 기능 | 설명 |
|------|------|
| 주간 시간표 | 요일/시간대별 반 배치 |
| 교실 배정 | 교실-시간대 충돌 방지 |
| 강사 스케줄 | 강사-시간대 충돌 방지 |
| 보강 스케줄 | 결석 학생 보강 일정 관리 |
| 시간표 변경 이력 | 변경 사유 및 히스토리 |

### 5.3 수강 상태 lifecycle

```
신규 문의
  → 레벨 테스트
  → 반 배정
  → 수강 등록 (ACTIVE)
  → 수강 중...
  ├→ 휴원 (ON_LEAVE) → 복원 (ACTIVE) 또는 퇴원
  ├→ 반 이동 (레벨 변경)
  └→ 퇴원 (WITHDRAWN) → 재등록 가능
```

## 6. 학부모 소통 상세

### 6.1 소통 채널 우선순위

| 순위 | 채널 | 용도 |
|------|------|------|
| 1 | **카카오 알림톡** | 출결 알림, 수납 알림, 공지 — 도달률 최고, 비용 저렴 |
| 2 | **앱 푸시 알림** | 즉시 알림 (알림장, 메시지, 성적) |
| 3 | **앱 내 메시지** | 1:1 상세 소통, 상담 내용 기록 |
| 4 | **SMS** | 알림톡 실패 시 대체, 긴급 공지 |

### 6.2 알림장 기능

| 항목 | 설명 |
|------|------|
| 일일 알림장 | 오늘 수업 내용, 숙제, 특이사항 |
| 주간 알림장 | 주간 학습 요약, 다음 주 계획 |
| 첨부 파일 | 수업 사진, 학습 자료, 시험지 |
| 반별/개별 발송 | 대상 지정 가능 |
| 읽음 확인 | 학부모 열람 여부 추적 |

### 6.3 학부모가 앱에서 확인하고 싶은 것 (중요도순)

1. **우리 아이가 학원에 잘 도착했는지** (출결 알림)
2. **오늘 뭘 배웠는지** (알림장)
3. **성적이 오르고 있는지** (성적 추이 그래프)
4. **단어를 얼마나 외우고 있는지** (어휘 테스트 결과)
5. **숙제를 했는지** (숙제 완료 현황)
6. **다음 달 수강료** (수납 내역)
7. **언제 다음 레벨로 올라가는지** (레벨 진척도)

## 7. 성적/학습 관리 상세

### 7.1 시험 유형

| 유형 | 빈도 | 설명 |
|------|------|------|
| 단어 테스트 | 매 수업 | 해당 단원 어휘 확인 |
| 단원 테스트 | 격주~월 | 교재 단원 완료 후 |
| 월말 테스트 | 월 1회 | 월간 학습 종합 평가 |
| 레벨업 테스트 | 분기 | 레벨 승급 여부 판단 |
| 모의고사 | 비정기 | 내신/수능 대비 |

### 7.2 성적 데이터 구조

| 항목 | 설명 |
|------|------|
| 총점/평균 | 시험별 점수 |
| 영역별 점수 | Reading, Listening, Writing, Speaking, Vocabulary, Grammar |
| 반 내 순위 | 반 평균 대비 위치 |
| 추이 그래프 | 시간 경과에 따른 점수 변화 |
| 오답 분석 | 틀린 문제 유형/영역 패턴 |
| 취약 단원 | 반복적으로 오답이 나오는 영역 |

### 7.3 학부모 리포트에 포함할 내용

```
[월간 학습 리포트]

1. 이번 달 학습 내용 요약
   - 진행한 교재/단원
   - 주요 학습 주제

2. 시험 결과
   - 각 시험 점수 + 반 평균 대비
   - 영역별 점수 비교 (레이더 차트)

3. 강점/약점 분석
   - 잘하는 영역 (칭찬 포인트)
   - 보완이 필요한 영역 (구체적 조언)

4. 출석 현황
   - 출석률, 결석 횟수

5. 숙제 현황
   - 완료율, 미제출 횟수

6. 어휘 학습
   - 단어 테스트 평균, 외운 단어 수

7. 교사 코멘트
   - 수업 태도, 발전 포인트, 권장 사항

8. 다음 달 계획
   - 학습 목표, 주안점
```

## 8. 상담 관리 상세

### 8.1 상담 유형

| 유형 | 설명 | 주요 기록 사항 |
|------|------|---------------|

| 레벨 테스트 | 배치 시험 상담 | 테스트 결과, 추천 반, 학부모 반응 |

| 학생 상담 | 학생 직접 | 학습 고민, 동기부여, 진로 |

| 퇴원 상담 | 퇴원 의사 표시 시 | 퇴원 사유, 만류 시도, 결과 |

- 상담 일시
- 상담 유형

- 담당자
- 상담 내용 요약
- 조치 사항 / follow-up 일정
- 상담 결과 (등록, 유지, 퇴원, 보류 등)

### 8.3 CRM 연계

```
신규 문의 상담
  → follow-up 알림 (3일 후)
  → 체험수업 예약
  → 체험 후 follow-up (1일 후)
  → 등록 전환 or 미전환 사유 기록
```

## 11. 마케팅/CRM 상세

### 11.1 신규 학생 유입 퍼널

```
[인지] 온라인 검색 / 소개 / 전단 / SNS
    ↓
[관심] 학원 홈페이지 / 블로그 방문
    ↓
[문의] 전화/방문/온라인 상담 신청
    ↓ ← 여기서부터 CRM 추적 시작
[상담] 상담 진행 + 레벨 테스트
    ↓
[체험] 체험수업 참여
    ↓
[등록] 수강 등록 + 결제
    ↓
[유지] 재원생 관리 + 만족도 관리
    ↓
[추천] 만족 → 지인 소개
```

### 11.2 CRM 기능

| 기능 | 설명 |
|------|------|
| **리드 관리** | 문의 접수 → 상태 추적 (신규/상담예약/상담완료/체험예약/체험완료/등록/미전환) |
| **자동 follow-up** | 상담 3일 후, 체험 1일 후, 미전환 7일 후 자동 리마인더 |
| **전환율 분석** | 단계별 전환율 (문의→상담 80%, 상담→체험 60%, 체험→등록 70% 등) |
| **유입 경로 추적** | 소개, 검색, SNS, 전단 — 채널별 효율 분석 |
| **추천인 관리** | 소개 학생 추적, 보상 자동 지급 (수강료 할인 등) |

### 11.3 퇴원/휴원 관리

| 기능 | 설명 |
|------|------|
| 퇴원 사유 분류 | 이사, 학업 부담, 강사 불만, 효과 미흡, 경제적 이유, 진학, 기타 |
| 퇴원 방지 알림 | 출석률 하락 + 성적 정체 시 이탈 위험 경고 |
| 휴원 관리 | 휴원 기간, 복원 예정일, 복원 유도 연락 |
| 재등록 캠페인 | 퇴원 6개월 후 재등록 유도 메시지 |
